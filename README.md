I write backend systems in **Go** and **Python** by day, and build things that probably didn't need to exist by night. Currently losing to OSTEP, learning BubbleTea TUIs, writing a hot reloader in Go, and assembling a split keyboard — which has so far been an expensive lesson in patience.

Tech Lead at **Hitachi Vantara**, based in **Pune, India**. Strong opinions on code readability. I got into distributed systems because I wanted to understand why things break in interesting ways. TUIs are an underrated art form and I will die on that hill.

**TUI with BubbleTea** — Learned TUI development the hard way. Turns out making a terminal app look good is a full-time job, and I already have one of those.
